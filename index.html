Some guidelines for beautiful and effective designs

  •   Use an 8 px grid, ideal for website layout design
  •   For icon design, use 1 px grid
  •   Short paragraphs 100â€“200 words (text line is 50 - 75 characters) with white space between paragraphs
  •   Use typography, colors, and spacing to guide visitors to information


  •   Provide ways to reveal more information gradually (details-HTML), but maintain the website pattern hierarchy
  •   Maintain the maximum 2 fonts on your website or graphics; one for big text and one for small text
  •   Text hierarchy-from large text to small text (up - bottom page)
  •   Colors - from light tones to dark shades (up - bottom page)

  •   Z pattern to create visuals (left to right, top to bottom reading), home pages, little content
  •   F pattern-pages with more text, starts reading from the top, concentrating on the content
  •   The human eye is all the time looking for patterns, making use of the same design blocks or elements, creating pattern navigation.

  •   SEO check on page
  •   W3C validation
  •   Use white space as much as possible, to enhance readability
  •   Show, by colors or graphics, the elements that shoud stand up, so that the user can concentrate on them

  •   Large space between text and visuals
  •   White space around an element
  •   Color contrast - AA or AAA
  •   Can also use Golden ratio in designing

  •   Use the circles from Golden ratio in making logos
  •   Use the Fibonacci sequence in making color palettes - RGB for example (40,80,100): a- 40 is the chosen number Red, b - double it, makes 80 for G, c - assure the last number for B is the sum of the prevoius numbers, so 120 is 40+80). Also, one can use HSB hue, saturation, brightness for making beautiful colors.
  •   Use the same color for all clickable buttons

  •   Multiply the base font size of 16px, for example, by the golden ratio of 1.618 and get multiples for your heading sizes. (16 px - base, 16px x 1.618 = 26 px - rounded), for the next font size use the previous font size obtained 26 px, so 26 px - base, 26px x 1.618 = 42 px - rounded and so on...)
  •   Use soft colors for information and background (like blue) and bold colors for CTA buttons (like orange)
  •   Font - line height 1.5, or 150%, the bigger the heading, the smaller the line height
  •   Use subtle color change and font weight to indicate hierarchy

  •   If the distance between title and text is 16px, the distance on top of them must be 32px, that means x2
  •   Spacing between letters minimum x0.12 times the font size
  •   Spacing between words minimum x0.16 times the font size
  •   Make use of the 6:3:1 Rule, 60% main color, 30% secondary color, 10% accent

  •   How to calculate height - golden ration: 960-pixel width layout divided by 1.618. The result is 594px, representing the height of the layout
  •   Check the design in grayscale to see how it looks and if it needs improvements
  •   Can use this UI for making a color theme: primary 500, secondary 200, third 700
  •   Use more saturation for minimalist colors

  •   A saturated color is more vibrant when it is used with different, less saturated colors
  •   Any two colors are harmonious when one or both contain some of the other
  •   The Fibonacci sequence is a series of numbers where each number is equal to the sum of the previous two numbers in the sequence - 3, 5, 8
  •   Fibonacci rule: add the two previous values so you can get the next one
